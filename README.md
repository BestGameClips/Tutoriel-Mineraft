# 📖 Présentation

**MineTutorials** est un projet web créé pour aider les joueurs Minecraft à apprendre facilement :

🎮 Minecraft Java Edition  
🟩 Minecraft Bedrock Edition  
🧩 Installation de mods  
🌍 Création de serveurs  
🔌 Plugins Minecraft  
🚀 Optimisation des performances  

Le site possède une interface moderne inspirée du **Glassmorphism**, avec un style proche des interfaces Apple.
