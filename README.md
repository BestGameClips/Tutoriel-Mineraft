# 📖 Présentation

## Qu'est-ce que MineTutorials ?

**MineTutorials** est un projet web ayant pour objectif de créer une plateforme regroupant des tutoriels Minecraft.

Le but est simple :

> Permettre aux joueurs débutants comme avancés de trouver rapidement des guides clairs, modernes et détaillés.

Le projet couvre :

🎮 Minecraft Java Edition  
🟩 Minecraft Bedrock Edition  
🧩 Mods  
🌍 Serveurs  
🔌 Plugins  
🚀 Optimisation
